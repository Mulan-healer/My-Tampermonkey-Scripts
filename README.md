# 🛠️ My Tampermonkey Scripts

这里收藏了我开发的所有油猴脚本，旨在提升 Web 端的各种交互体验。

---

## 🚀 重点推荐

### 📺 OpenList to MPV 增强脚本
**功能：**
* 一键从 OpenList 调用本地 **MPV** 播放器。
* **内置观看历史记录**，补齐原生缺失的进度保存功能。
* 极简 UI 交互，画质党福音。

👉 [查看安装与配置说明](./OpenList-MPV-Enhancer)

## 🛠️ 如何使用
1. 确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. 点击列表中的【安装】链接。
3. 部分脚本（如 MPV 调用）可能需要配合本地后端脚本，请参阅具体脚本目录下的文档。
